Single Pauli String Measurements
All circuit measurements are performed in the Z pauli basis: If we want to evaluate a circuit in the expectation of the form ⟨Z0Z1Z3...Zn⟩\langle Z_0Z_1Z_3...Z_n\rangle⟨Z0​Z1​Z3​...Zn​⟩﻿, simply measuring in the computational basis and computing the expectation of the measurement from the measurement distribution is sufficient.
Pauli ZZZ﻿ basis measurements and expectations:
We start by computing the expectation of a single qubit:
Measurement produces buckets of expectations and shots. If we focus only on a single qubit, this is something like a dictionary: {’0’: 55, ‘1’: 45}
Computing the observable ⟨Z0⟩=Σi(λiPi) \langle Z_0 \rangle = \Sigma_i (\lambda_iP_i)⟨Z0​⟩=Σi​(λi​Pi​)﻿, where λi\lambda_iλi​﻿ is the ithi^{th}ith﻿ eigenvalue of the Pauli matrix we are evaluating, and PiP_iPi​﻿ is the probability of the state measurement.
The eigenvalues of Z0Z_0Z0​﻿ are +1 and -1, with basis states ∣0⟩\ket{0}∣0⟩﻿ and ∣1⟩\ket{1}∣1⟩﻿, respectively.
Example:
Z0=∣0⟩⟨0∣−∣1⟩⟨1∣Z_0 = \ket{0}\bra{0} - \ket{1}\bra{1}Z0​=∣0⟩⟨0∣−∣1⟩⟨1∣﻿
The probability of measuring ∣0⟩\ket{0}∣0⟩﻿ is given by: ⟨ψ∣(∣0⟩⟨0∣)∣ψ⟩\bra{\psi}(\ket{0}\bra{0})\ket{\psi}⟨ψ∣(∣0⟩⟨0∣)∣ψ⟩﻿
So, when we want to measure Z0Z_0Z0​﻿, we take ⟨ψ∣Z0∣ψ⟩=⟨ψ∣(∣0⟩⟨0∣−∣1⟩⟨1∣)∣ψ⟩=P0−P1\bra{\psi}Z_0\ket{\psi} = \bra{\psi}(\ket{0}\bra{0} - \ket{1}\bra{1})\ket{\psi} = P_0 - P_1⟨ψ∣Z0​∣ψ⟩=⟨ψ∣(∣0⟩⟨0∣−∣1⟩⟨1∣)∣ψ⟩=P0​−P1​﻿
For multiple qubits, we take the same form, but with more eigenvalues:
Z0Z1Z_0Z_1Z0​Z1​﻿ can be decomposed as: ∣00⟩⟨00∣−∣01⟩⟨01∣−∣10⟩⟨10∣+∣11⟩⟨11∣\ket{00}\bra{00} - \ket{01}\bra{01} - \ket{10}\bra{10} +\ket{11}\bra{11}∣00⟩⟨00∣−∣01⟩⟨01∣−∣10⟩⟨10∣+∣11⟩⟨11∣﻿\
And so can be evaluated using P00+P11−P01−P10P_{00} + P_{11} - P_{01} - P_{10}P00​+P11​−P01​−P10​﻿
Measuring in different bases requires rotating the from the Z basis into the desired basis. Take the simple case of an ⟨X0⟩\langle X_0 \rangle⟨X0​⟩﻿ measurement. Applying a Hadamard to the end of a circuit and then measuring in the computational basis produces expectations if the type:
⟨ψ∣H(∣0⟩⟨0∣)H∣ψ⟩→⟨ψ∣(∣+⟩⟨+∣)∣ψ⟩\bra{\psi}H(\ket{0}\bra{0})H\ket{\psi} \rightarrow \bra{\psi}(\ket{+}\bra{+})\ket{\psi}⟨ψ∣H(∣0⟩⟨0∣)H∣ψ⟩→⟨ψ∣(∣+⟩⟨+∣)∣ψ⟩﻿
⟨ψ∣H(∣1⟩⟨1∣)H∣ψ⟩→⟨ψ∣(∣−⟩⟨−∣)∣ψ⟩\bra{\psi}H(\ket{1}\bra{1})H\ket{\psi} \rightarrow \bra{\psi}(\ket{-}\bra{-})\ket{\psi}⟨ψ∣H(∣1⟩⟨1∣)H∣ψ⟩→⟨ψ∣(∣−⟩⟨−∣)∣ψ⟩﻿
So, we can evaluate X=∣+⟩⟨+∣−∣−⟩⟨−∣X = \ket{+}\bra{+} - \ket{-}\bra{-}X=∣+⟩⟨+∣−∣−⟩⟨−∣﻿ by simply computing ⟨Z⟩\langle Z \rangle ⟨Z⟩﻿ in this new basis
Measuring in the YYY﻿ basis requires applying an Rx(π/2)R_x(\pi/2)Rx​(π/2)﻿ gate, which can be generated using an S†HS^{\dagger}HS†H﻿ addendum to the end of the circuit
Parallel Measurements:
In general, we can compute parallel observables for two cases:
Each site of the Pauli string commutes:
In this case, we simply rotate appropriately to the ZZ basis for each string:
XIIZ→ZIIZXIIZ \rightarrow ZIIZXIIZ→ZIIZ﻿
XYIZ→ZZIZXYIZ \rightarrow ZZIZXYIZ→ZZIZ﻿
XYXI→ZZZIXYXI \rightarrow ZZZIXYXI→ZZZI﻿
by applying H0S1H1H2H_0S_1H_1H_2H0​S1​H1​H2​﻿ before measuring
The Pauli strings commute, but not locally:
In this case, we need to find a basis in which these strings are diagonal. This can be performed using Clifford circuits, but is not necessarily easy to do
Useful Reference: O(n^3) measurements, IEEE
