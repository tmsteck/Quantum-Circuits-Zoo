The Clifford Group is the set of operations which map Pauli group elements to Pauli group elements under conjugation:
Definition (Clifford Group): A unitary matrix CCC﻿ is in the Clifford group if CPCC\mathcal{P}CCPC﻿ is an element of the Pauli Group
Definition (Pauli Group): GnG_nGn​﻿ is the group generated by all Pauli Strings Pn\mathcal{P}_nPn​﻿ under multiplication
A standard method for generating the Clifford group is using the SSS﻿, CXCXCX﻿, and HHH﻿ gates.
